Esta é a minha Landing Page profissional, desenvolvida como um ecossistema visual para apresentar meus projetos em Fullstack e Ciência da Computação. O projeto foca em alta performance, interatividade via DOM e design moderno.

---

## 🛠️ Tecnologias e Arquitetura

Para construir esta interface, foquei na separação de interesses e em uma experiência de usuário (UX) reativa:

- **HTML5 Semântico:** Estruturação de dados utilizando seções e IDs únicos para manipulação precisa via Script.
- **CSS3 Moderno:** 
  - **Bento Grid Layout:** Uso de CSS Grid para um design responsivo e organizado.
  - **Glassmorphism:** Efeitos de desfoque (`backdrop-filter`) e transparências.
  - **Aurora Effect:** Animações de gradiente fluído no background.
- **JavaScript (Engenharia do DOM):**
  - **Seleção Dinâmica:** Uso de `getElementById` e `querySelectorAll` para controle de fluxo.
  - **Manipulação de Atributos:** Injeção de conteúdo em tempo real e controle de estilo inline.
  - **Gerenciamento de Eventos:** Implementação de `addEventListener` para interações de `mouseover`, `mouseout` e `click`.
  - **Assincronia:** Uso de `setTimeout` para controle de vida de elementos na interface.

---

## ✨ Funcionalidades Destacadas

1.  **Interatividade Reativa:** Links que mudam de estado e texto ao passar o mouse, guiando a navegação do usuário.
2.  **Sistema de Feedback Visual:** Efeito de iluminação que segue o cursor do mouse através dos cards da grid.
3.  **Grid Adaptativa:** Seção de projetos técnica que se ajusta automaticamente ao tamanho da tela (Responsividade Mobile-First).
4.  **Injeção de Conteúdo Dinâmico:** Mensagens de boas-vindas controladas programaticamente via JavaScript.
